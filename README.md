LHIRES III Spectrograph Simulator

A browser-based 3D simulator of the LHIRES III spectrograph built with Three.js.
The simulator visualizes how light travels through a typical Littrow spectrograph configuration, allowing users to interactively change optical parameters and observe how they affect the spectrum.

This tool is primarily intended for teaching and conceptual understanding of spectrograph optics, especially for students working with instruments like the LHIRES III used in amateur and educational observatories.

Features
3D Optical Bench Visualization

The simulator renders the main optical components of the spectrograph in a fully interactive 3D scene:

Input slit
Folding mirror
Collimator lens
Diffraction grating
CCD detector

Users can rotate, pan, and zoom the scene to inspect the instrument geometry.
